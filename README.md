# 🧙 RuneScapeConsoleApp

A text-based RPG inspired by classic MMORPG progression systems.  
Built in C++ as a console application.

## 🎮 About The Game

Terminal Realms is a console-based RPG where players can:

- Train skills (Attack, Magic, Woodcutting)
- Fight enemies
- Complete quests
- Buy items from the Grand Exchange
- Manage inventory and gold
- Cast spells using runes
- Progress through levels
---

## 🛠 Built With

- C++
- Standard Library (iostream, string)
